# Pong Remake

A simpe remake of the popular game Pong. This project is not meant to be perfect and is just for fun.

Made using C++ and the [SFML](https://www.sfml-dev.org/) library.

## How to build

To build this project you'll need `python` and the `CMake` tool installed.

Supmorted build types: Debug and Release
Supported lib types: Static and Shared

Run `python build.py build [build-type] [lib-type]` to build the project in Static Debug mode by default.

Run `python build.py run [build-type] [lib-type]` to build and run the project.
